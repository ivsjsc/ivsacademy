# IVS Celestech - Quick Start Guide

## For Users

### 1. Sign In
- Visit https://ivsacademy.edu.vn/Pages/ivs-celestech/
- Click "Sign in with Google"
- Authorize the app

### 2. Create Request
1. Click "Create Request" tab
2. Fill in the form:
   - **Title**: What you need (e.g., "Translate API Documentation")
   - **Description**: More details about your request
   - **Task Type**: Choose what you need (Translate, Summarize, Generate Content, Analyze)
   - **Content**: The text/data to process
   - **Target Language** (if translating): Choose language
3. Click "Submit Request"
4. Wait for admin approval

### 3. Track Progress
- Click "My Requests" tab
- See status of all your requests
- Status meanings:
  - 🟡 **Pending Approval**: Admin hasn't reviewed yet
  - 🔵 **Approved**: Ready to process
  - ⚙️ **Processing**: AI is working on it
  - ✅ **Completed**: Ready to view/download result
  - ❌ **Rejected**: Admin declined; check notes

---

## For Admins

### 1. Sign In
- Use your admin email (e.g., admin@ivsacademy.edu.vn)
- "Admin Panel" tab will appear

### 2. Review Requests
1. Click "Admin Panel" tab
2. See all pending requests
3. Read request details

### 3. Approve/Reject
**To Approve:**
1. Click "Approve & Process" button
2. AI will automatically process
3. User gets notified when done

**To Reject:**
1. Click "Reject" button
2. Enter rejection reason
3. User will see reason

**To Add Notes:**
1. Click "Add Notes" button
2. Type your comment
3. User will see notes
