IVS - Verified ID Manifest Generator

This tool creates a small JSON manifest describing a Verified ID for an employee and produces a data: URL (or downloadable .json) you can host or pass to verification systems.

Notes: This is a local generator only. For production, sign manifests and host them on a secure endpoint. The page does not transmit data to any external server.

Usage Instructions

  1. Enter the Employee ID (required) and optional details like name, title, or extra JSON payload.
  2. Click "Generate Manifest URL" to create the manifest preview and data URL.
  3. Use the "Copy URL" button to copy the data: URL for use in verification systems.
  4. Alternatively, click "Download Manifest" to save the JSON file locally.
  5. For external verification, pass the manifest URL or file to your Verified ID tooling (e.g., Microsoft Authenticator or custom verifier).
  6. In production, host the manifest on a secure HTTPS endpoint and sign it with a cryptographic key for authenticity.
