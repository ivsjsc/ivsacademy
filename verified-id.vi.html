IVS - Trình Tạo Manifest ID Đã Xác Minh

Công cụ này tạo một manifest JSON nhỏ mô tả ID Đã Xác Minh cho nhân viên và tạo ra một URL data: (hoặc .json có thể tải xuống) mà bạn có thể lưu trữ hoặc truyền đến hệ thống xác minh.

Lưu ý: Đây chỉ là trình tạo cục bộ. Đối với sản xuất, ký manifest và lưu trữ chúng trên điểm cuối bảo mật. Trang này không truyền dữ liệu đến bất kỳ máy chủ bên ngoài nào.

Hướng Dẫn Sử Dụng

  1. Nhập ID Nhân Viên (bắt buộc) và các chi tiết tùy chọn như tên, chức vụ hoặc dữ liệu JSON bổ sung.
  2. Nhấp "Tạo URL Manifest" để tạo xem trước manifest và URL dữ liệu.
  3. Sử dụng nút "Sao Chép URL" để sao chép URL data: để sử dụng trong hệ thống xác minh.
  4. Ngoài ra, nhấp "Tải Xuống Manifest" để lưu tệp JSON cục bộ.
  5. Đối với xác minh bên ngoài, truyền URL manifest hoặc tệp đến công cụ Verified ID của bạn (ví dụ: Microsoft Authenticator hoặc trình xác minh tùy chỉnh).
  6. Trong sản xuất, lưu trữ manifest trên điểm cuối HTTPS bảo mật và ký nó bằng khóa mật mã để đảm bảo tính xác thực.
